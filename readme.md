# b.gif.ly

v2.0*beta*

Welcome to gif.ly. Upload animated gifs and share with anyone. This is
an [open source project](https://github.com/wookiehangover/node-gifly)
and was built with [Node.js](http://nodejs.org) and
[Redis](http://redis.io).
